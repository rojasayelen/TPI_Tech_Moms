## Cómo Generar los Diagramas

### Opción 1: Mermaid (Recomendado)
1. Copiar el código Mermaid de `ARCHITECTURE.md`
2. Usar [Mermaid Live Editor](https://mermaid.live/)
3. Exportar como PNG
4. Guardar en esta carpeta

### Opción 2: Draw.io
1. Ir a [draw.io](https://app.diagrams.net/)
2. Crear diagrama basado en la documentación
3. Exportar como PNG
4. Guardar en esta carpeta

### Opción 3: Lucidchart
1. Usar plantillas de arquitectura
2. Seguir la estructura documentada
3. Exportar como PNG

### Opción 4: Canva
1. Usar plantillas de diagramas técnicos
2. Personalizar con los componentes del sistema
3. Descargar como PNG

## Especificaciones de Imágenes

- **Formato:** PNG
- **Resolución:** Mínimo 1200x800px
- **Calidad:** Alta definición
- **Fondo:** Blanco o transparente
- **Texto:** Legible en tamaño pequeño

## Actualización de Diagramas

Los diagramas deben actualizarse cuando:
- Se agreguen nuevos componentes
- Cambie la arquitectura
- Se modifique el pipeline CI/CD
- Se actualicen las tecnologías

---

**Nota:** Los archivos PNG reales deben ser generados manualmente usando las herramientas mencionadas arriba.
